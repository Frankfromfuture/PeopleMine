# Changelog

All notable changes to this project will be documented in this file.

The format loosely follows Keep a Changelog, adapted for the current demo-stage workflow.

---

## [0.4.0] - 2026-04-06

### Added
- Expanded marketing site structure with dedicated `product`, `solutions`, and `pricing` pages.
- Post-login workspace navigation for dashboard, profile, contacts, companies, journey, and several upcoming modules.
- Company database flows, including company creation, listing, detail routing, and company-universe entry.
- More complete contact modeling for personal, company, and social-profile attributes.
- Additional API routes for companies, network graph data, table views, and development utilities.
- New UI building blocks and visualization-oriented components for dashboard and graph-based experiences.
- Initial repository changelog.

### Changed
- Project package name updated to `peoplemine`.
- Project version bumped from `0.1.0` to `0.4.0`.
- README rewritten to reflect the current product scope instead of the early-stage concept demo.
- Contact system evolved from the original lightweight role/animal model toward a richer archetype-driven schema.
- Schema and app structure expanded to support company relations, journey analysis, and broader data capture.

### Notes
- This version represents the transition from an early concept demo into a runnable multi-module product prototype.
- Several pages are intentionally present as structural placeholders while the product architecture continues to expand.
