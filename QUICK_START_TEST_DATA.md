# 快速测试数据生成器 — 使用指南 🧪

## 📍 在哪里找到？

在首页（Dashboard）向下滚动，找到 **「快速测试数据」** 卡片。

外观：紫色渐变背景，左上角有 🧪 图标。

---

## 🎯 三步快速开始

### 第 1 步：设置参数

```
生成人员数量：[10]   个
标签波动性：   [50]  %
```

- **数量**：1-500 之间任选
- **波动性**：
  - 0% = 标签与行业完全相关（现实感强）
  - 50% = 混合（推荐用于一般测试）
  - 100% = 标签完全随机（压力测试）

### 第 2 步：点击「✨ 一键生成」

系统会自动：
1. 生成随机中文名字
2. 随机分配行业、公司、职位
3. 生成相关或随机的标签
4. 分配能量值、温度、信任度等
5. 按比例分配关系角色
6. 保存到人脉数据库

### 第 3 步：查看结果

页面自动刷新，你会看到新增的联系人出现在：
- 首页「待跟进人脉」列表
- 「人脉数据库」(/contacts)
- 人脉航程分析中

## ⚙️ 各项参数的作用

### 生成数量

| 数量 | 用途 | 时长 | 特点 |
|------|------|------|------|
| 10-20 | 快速体验 | < 2 秒 | 快，足以演示功能 |
| 30-50 | 一般测试 | 2-5 秒 | 均衡 |
| 100+ | 压力测试 | 10-30 秒 | 验证性能 |
| 500 | 极限测试 | 30-60 秒 | 系统上限 |

### 标签波动性

| 波动性 | 标签特点 | 使用场景 | 航程结果 |
|--------|---------|---------|---------|
| 0% | 所有标签与行业相关 | 现实感测试 | 路径清晰，建议精准 |
| 25% | 大多数相关，少量随机 | 一般推荐 | 现实与随机平衡 |
| 50% | 50% 相关 + 50% 随机 | 通用测试 | 多样性好 |
| 75% | 大多数随机，少量相关 | 难度测试 | 算法鲁棒性验证 |
| 100% | 完全随机 | 压力测试 | 验证极端情况 |

## ❌ 常见问题

**Q: 数据会重复吗？**
A: 名字理论上可能重复，但概率 < 0.1%。其他数据完全随机生成，不会重复。

**Q: 清除后能恢复吗？**
A: 不能。清除是不可逆操作。请谨慎点击！

**Q: 能自定义生成的数据吗？**
A: 暂时不能。当前是完全随机。后续可以扩展为自定义公司、标签、行业等。

**Q: 关系边（人脉之间的连接）也会生成吗？**
A: 不会。当前只生成 Contact 记录。可以手动在数据库中添加关系。

**Q: 生成 500 个需要多长时间？**
A: 通常 30-60 秒。取决于数据库性能。

**Q: 真实数据和测试数据能混在一起吗？**
A: 完全可以。生成的测试数据和手动添加的真实数据没有区别。

